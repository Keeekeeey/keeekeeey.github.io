# personal-portfolio

A unique portfolio website featuring a retro desktop interface with draggable windows and interactive desktop icons. Built with React and Next.js for a nostalgic yet modern user experience.
## Features

- **Draggable Windows** - Fully interactive windows that can be moved around the screen
- **Desktop Icons** - Click to open different sections (About, Projects, Contact, etc.)
- **Retro Desktop UI** - Nostalgic desktop environment aesthetic
- **Responsive Design** - Adapts to different screen sizes
- **Fast & Modern** - Built with Next.js for optimal performance

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **UI Library**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: JavaScript/TypeScript
